# Brainstorming Skill (Brain Jazz)

## Опис
Цей skill надає AI асистенту можливість проводити креативні сесії мозкового штурму з використанням різних технік.

## Можливості
- Генерація ідей за різними техніками
- Структурований мозковий штурм
- Асоціативне мислення
- Креативне вирішення проблем

## Техніки мозкового штурму

### 1. Класичний мозковий штурм
```
Проведи мозковий штурм на тему [тема]
```
- Генерація максимальної кількості ідей
- Без критики на етапі генерації
- Групування та пріоритезація

### 2. SCAMPER
```
Застосуй SCAMPER до [об'єкт/проблема]
```
- **S**ubstitute - Що можна замінити?
- **C**ombine - Що можна об'єднати?
- **A**dapt - Що можна адаптувати?
- **M**odify - Що можна модифікувати?
- **P**ut to other use - Як можна використати інакше?
- **E**liminate - Що можна прибрати?
- **R**everse - Що можна перевернути?

### 3. Шість капелюхів мислення
```
Проаналізуй [тема] через шість капелюхів
```
- ⚪ **Білий** - Факти та дані
- 🔴 **Червоний** - Емоції та почуття
- ⚫ **Чорний** - Ризики та критика
- 🟡 **Жовтий** - Переваги та можливості
- 🟢 **Зелений** - Креатив та альтернативи
- 🔵 **Синій** - Процес та організація

### 4. Ментальна карта
```
Створи ментальну карту для [тема]
```
- Центральна ідея
- Основні гілки
- Деталізація

### 5. Зворотний мозковий штурм
```
Як можна ЗАВДАТИ проблему [тема]?
```
- Генерація способів створення проблеми
- Інверсія для знаходження рішень

## Формат результатів
```markdown
## Мозковий штурм: [Тема]

### Техніка: [Назва техніки]

### Ідеї:
1. [Ідея 1]
2. [Ідея 2]
...

### Топ-3 ідеї:
1. [Найкраща ідея]
2. [Друга найкраща]
3. [Третя найкраща]

### Наступні кроки:
- [ ] [Дія 1]
- [ ] [Дія 2]
```

## Інтеграція з Second Brain
Результати мозкового штурму можна зберегти в:
- `MEMORY.md` - для важливих ідей
- `memory/YYYY-MM-DD.md` - для щоденних сесій
