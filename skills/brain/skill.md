# Second Brain Skill

## Опис
Цей skill надає AI асистенту доступ до вашої "другої пам'яті" - структурованої системи для зберігання та організації інформації.

## Можливості
- Збереження та отримання інформації з MEMORY.md
- Створення щоденних записів у memory/
- Організація знань за категоріями
- Синхронізація з GitHub для доступу з різних пристроїв

## Структура файлів
```
brain/
├── MEMORY.md          # Основна пам'ять
├── memory/            # Щоденні записи
│   └── YYYY-MM-DD.md  # Файл за день
├── skills/            # Навички агентів
│   ├── brain/         # Second Brain skill
│   └── brain-jazz/    # Brainstorming skill
└── README.md          # Інструкції
```

## Команди
### Зберегти інформацію
```
Збережи [інформація] в [категорія]
```

### Отримати інформацію
```
Що я знаю про [тема]?
```

### Створити запис
```
Створи запис за [дата]
```

## Синхронізація
### Pull (отримати зміни):
```bash
git pull origin main
```

### Push (відправити зміни):
```bash
git add .
git commit -m "Update memory"
git push origin main
```

## Використання
1. Вкажи AI асистенту використати цей skill
2. AI зможе читати та оновлювати MEMORY.md
3. AI зможе створювати щоденні записи
4. Зміни синхронізуються з GitHub
