# Screenshot Reader Skill

## Опис
Цей skill надає AI асистенту можливість аналізувати скріншоти та зображення з використанням Google Gemini Vision API. Ідеально підходить для розпізнавання тексту, аналізу UI помилок, та витягування даних з візуальних матеріалів.

## Можливості
- Аналіз скріншотів програм та веб-сторінок
- Розпізнавання тексту на зображеннях (OCR)
- Опис вмісту зображень
- Відповіді на запитання про зображення
- Витягування даних з візуальних матеріалів
- Аналіз помилок та діагностичної інформації
- Розпізнавання коду на скріншотах

Ця функція перетворює список словників у словник з ключами 'id'.

## Використання в Slack

### Спосіб 1: Надіслати зображення напряму
1. У Slack чаті з ботом Jin надішліть зображення (фото або скріншот)
2. Додайте текстовий запит у повідомленні:
   ```
   Проаналізуй цей скріншот
   ```
3. Бот автоматично використає vision модель для аналізу

### Спосіб 2: Використати команду /skill
```
/skill screenshot-reader [опис що потрібно зробити]
```

### Приклади для Slack:
- Надішліть скріншот помилки з текстом: `Що не так на цьому скріншоті?`
- Надішліть фото коду: `Розпізнай цей код`
- Надішліть скріншот UI: `Опиши інтерфейс`

### Важливо для Slack:
- Бот Jin повинен мати дозвіл `files:read` для читання зображень
- Зображення автоматично конвертуються в base64 для API
- Максимальний розмір файлу: 20MB

## Обмеження
- Максимальний розмір зображення: 20MB
- Підтримувані формати: PNG, JPEG, WebP, GIF, HEIC, HEIF
- Максимальна кількість зображень за запит: 3,600
- Контекстне вікно: 1,000,000 токенів

## Поради для кращих результатів

### Якість зображення
- Використовуйте чіткі, не розмиті зображення
- Переконайтеся, що текст добре читається
- Уникайте зайвих тіней та відблисків

### Формулювання запитів
- Будьте конкретні в тому, що потрібно витягнути
- Вказуйте формат результату (JSON, таблиця, список)
- Для коду вказуйте мову програмування

### Складні зображення
- Для великих скріншотів вказуйте область інтересу
- Для діаграм просіть описати кожен елемент окремо
- Для форм просіть витягнути конкретні поля
