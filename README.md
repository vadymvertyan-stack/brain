# Brain

Second brain notes, skills and plans for AI assistant integration.

## Structure

- `brain/` - Skills for OpenClaw/Jin assistant
- `memory/` - Notes and memories
- `plans/` - Project plans and research
- `*.md` - Documentation

## Skills

Skills are used by Jin/OpenClaw AI assistant to perform tasks:
- `brain-jazz` - Music and jazz related queries
- `context-compression` - Context compression for long conversations
- `large-plans` - Planning for complex tasks
- `screenshot-reader` - Screenshot analysis
- `second-brain` - Note management
- `task-manager` - Task management

## Database

Uses SQLite with vector search (sqlite-vec) for semantic search.
